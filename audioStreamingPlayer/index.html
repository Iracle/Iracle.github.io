简单的网络流媒体播放下载

Posted on 2016-02-22

简介

AVFoundation.framework 中的 AVAudioPlayer 虽然有完整的播放控制和多种音频格式的支持，但它仅仅能播放本地音乐。对于网络流媒体是不提供支持的。那 iOS 中怎么实现播放网络流媒体呢？可以使用 AudioToolbox.framework 的音频队列服务 Audio Queue Services。至于原理，苹果有完整的文档和图例，请移步 这里 。当然就算你看明白原理了，也还有大量的工作要做，比如：控制播放状态，音频编码、解码，异常中断处理等等。所以要想完整的实现一个比较完整的网络流媒体播放器，是需要一定时间的。本文的目的不是这个，如果你对音频很有兴趣，这里顺便推荐一个博客 码农人生 ，讲了一些音频的基础知识。本文使用一个开源的框架 AFSoundManager 来实现网络流媒体的播放，AFSoundManager 是基于 AudioToolbox 和 AVFoundation 框架实现的网络流媒体播放库，同时也支持播放本地音乐和录音，可以去 AFSoundManager 的 Github 主页 查看更多信息。

总结

本文只是简单实现了播放逻辑，距离一个完善的音乐播放器需要完善的地方很多，大家有兴趣可以自行拓展一下，比如增加边播放边缓存的功能，设置 AudioSession 等等。此外，还可以用苹果提供的 AVPlayer 实现流媒体播放，通过拦截请求的方式实现音频数据缓存分析。希望本文对初学者有一些帮助。
