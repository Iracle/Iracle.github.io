Rao 的前世今生

Posted on 2016-11-24

前段时间突然对绕口令感兴趣，去 App Store 里找了一下，发现里面绕口令 APP 倒是不少，但是质量惨不忍睹，好多都是许久不更新的了，UI 设计还停留在上古世纪。实在选不出一个可以忍下心装到手机里面的，就萌生了自己写一个的想法。因为自己前面独自完成过 Gust Browser 所以对个人项目还是很轻车熟路。于是趁热打铁，定下了产品的风格。

  • 使用本地数据库，无需联网
  • 绕口令有不同的难度等级
  • 设计简洁优雅，交互自然
  • swift 2.3

然后用 Sketch 画了基本的原型图：



原型有了，可以开始编码了。绕口令的数据库是淘宝买的，要相信淘宝什么都买的到。用的原生 SQlite 来操作数据库，数据持久化层则用的是 RealmSwift 。Autolayout 选择 Snapkit ，最后使用了 storehouse 开源的动画库 Advance , Advance 是一个纯 Swift 库，和 Core Animation 和 pop 相比，Advance 提供了强大的 API 用于创建可交互式的动画。由于是个人项目，没有进度上面的要求，所以就慢慢写。写到一半，还用 RxSwift 重构了一次，随着对 RxSwift 的深入，感觉由于它对 Cocoa 框架的支持不全，对于需求复杂的产品，还是不能统一代码风格，反而会增加复杂度，遂弃之。期间苹果发布了 Swift 3.0 ，所以只得老老实实的迁移到 3.0😓。断断续续的写了三个月，终于在十一月提交审核，结果两天就通过了，苹果审核的效率什么时候变的这么高了。遥想今年四月份提交 Gust Browser 的时候被拒了一次，等了十多天才上架，那种不爽啊。由于这个 APP 很简单，本来想免费的，但想到自己没那么多精力做客服，就定了一个价，筛选掉大部分用户，只给真正需要的人。




恩，没什么想说的了。
