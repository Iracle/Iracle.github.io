使用 3D Touch 让你的 APP 锦上添花

发表于 2015-11-17   |   分类于 ai

简介

3D Touch 是苹果在 iPhone6s/pluse 上引入的一种全新的交互方式。这也许是手机交互发展史上继苹果 2007 年在 WWDC 上发布第一代 iPhone 之后最大的革新了。当时苹果抛弃传统的全键盘手机，发布了一款带有电容式触摸屏支持多点触控的全触屏手机 iPhone。自此，全触屏手机席卷全球，到现在我们都还处于全触屏手机的风潮中，你很难再去买一块带键盘的手机了（而那些想要同时兼顾键盘和触屏的手机厂商已经死了。对，我就是在黑诺基亚，我不是人）。多点触控是全触屏手机能够流行的一大基石，用户很方便通过 pinch 来缩放一张图片，我在 5 年前第一次在 iTouch（当时买不起 iPhone ）上放大一张图片那种操作快感带给我的喜悦我至今都记得。但在手机上无论是 pinch 还是滑动操作都是在一个二维平面上发生的，直到 iPhone6s/pluse 的到来。新一代 iPhone 可以感知我们的压力，并且根据压力的不同给用户不用的反馈，其原理是在液晶屏的下面增加了一块压力传感器。相信我，如果你有体验过 iPhone 的 3D Touch 你一定会爱上她的，反正用了一个月 6s 后我已经离不开她了。

除了苹果自家的应用已经支持 3D Touch 之外，苹果也开放了以下 3D Touch API。

  • Home Screen quick action
  • UIKit peek and pop
  • Web view peek and pop
  • UITouch force properties

使用这些 API 可以让我们的 APP 在交互上更上一层楼！

Home Screen quick action

Static quick actions 是在应用的 Info.plist 文件的 UIApplicationShortcutItems 数组中定义 静态快速选项。
